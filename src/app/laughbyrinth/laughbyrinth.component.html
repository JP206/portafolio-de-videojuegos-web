Laughbyrinth: Quest for the Legendary Joke

¿Can you get the legendary joke and escape the evil clown protecting it?

This game was developed for the 2024 Global Game Jam. The theme was "Make me laugh", this game aims to be a fun parody of terror games. I was the only programmer in my team.
